## OBJETIVO

Criar um sistema de  envio e recebimento de mensagens em tempo real utilizando electron com MQTT, API, React.



## PLANEJAMENTO

### ESCOPO 

- VAI TER
1. brokerMQTT - NodeJs
2. API - NodeJS
3. Web - React
4. Compilado em Electron - NodeJs
5. Persistencia de Dados - fs
6. Sincronia de Mensagens - brokerMQTT
7. Contatos - fs
8. Login - fs

- NÃO VAI TER
1. Criptografia
2. Banco de Dados
3. Segurança

### FERAMENTAS 

1. Backend - NodeJS 
2. Frontend - React
3. Comunicação - MQTT
4. BrokerMQTT - aedes
5. Api - express
6. Gerenciamento de Arquivos - fs
7. Desktop - electron
